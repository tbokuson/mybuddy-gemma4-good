## Description

MyBuddy is a private AI diary app for iPhone that turns small daily conversations and photos into editable diary entries.

This demo shows:

- Creating a personal AI buddy
- Customizing the buddy's look, tone, and distance
- Starting from a casual daily conversation instead of a blank diary page
- Attaching a photo from the day
- Using Gemma 4 for multimodal text-and-image reflection
- Generating a private diary entry
- Keeping diary entries organized over time

MyBuddy uses Gemma 4 E2B-it locally through llama.cpp, with image understanding via mmproj-F16 / mtmd / CLIP. After the initial model download, chats, photos, diary generation, and storage stay on the user's iPhone.

MyBuddy is a journaling and reflection app. It is not a medical, therapy, diagnostic, legal, or financial advice product.
